算法	类型	稳定性	最好	最坏	平均	空间复杂度	是否原地排序
Selection Sort	选择类	❌ 不稳定	O(n²)	O(n²)	O(n²)	O(1)	✅ 是
Insertion Sort	插入类	✅ 稳定	O(n)	O(n²)	O(n²)	O(1)	✅ 是
Bubble Sort	交换类	✅ 稳定	O(n)	O(n²)	O(n²)	O(1)	✅ 是
Merge Sort	分治类	✅ 稳定	O(n log n)	O(n log n)	O(n log n)	O(n)	❌ 否（需辅助数组）
Quick Sort	分治类	❌ 不稳定	O(n log n)	O(n²)	O(n log n)	O(log n)	✅ 是
Heap Sort	堆结构	❌ 不稳定	O(n log n)	O(n log n)	O(n log n)	O(1)	✅ 是
Radix Sort	非比较类	✅ 稳定	O(nk)	O(nk)	O(nk)	O(n + k)	❌ 否
